Developer Reference

FlagJS

Flag.js's functionality is conveniently located in a global object named FlagJS.

FlagJS.getFlag

function getFlag(flagName: string): string | undefined

This function returns a flag for the provided flag name.

Parameters
flagName

The name of the flag to retrieve. Possible flags include:

  • "black"
  • "chequered"
  • "crossed"
  • "ctf"
  • "inHole"
  • "pirate"
  • "rainbow"
  • "transgender"
  • "triangular"
  • "white"

Note: Some flags may appear incorrectly on older operating systems.
Returns

The flag, or undefined if flagName was not found.
